# cpo-results-collector

## Usage

```
usage: cpo-results-collector [-h] [-d ANALYSIS_DIR] [-a ASSEMBLER] [--noheader] [--output OUTPUT] [--plasmid-output PLASMID_OUTPUT]

options:
  -h, --help            show this help message and exit
  -d ANALYSIS_DIR, --analysis-dir ANALYSIS_DIR
  -a ASSEMBLER, --assembler ASSEMBLER
  --noheader
  --output OUTPUT
  --plasmid-output PLASMID_OUTPUT
```
